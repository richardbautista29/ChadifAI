# Fundamentals

This folder contains foundational practice scripts used during my transition into AI Engineering.

The goal of this section is to build a strong understanding of the core tools and concepts that power modern AI applications.

## Scope

The scripts here are intentionally small and focused. Each file demonstrates a specific concept rather than being a full project.

Topics covered include:

- NumPy (array operations, basic statistics)
- Pandas (data loading, filtering, aggregation)
- LLM Basics (API calls, prompt handling, structured output)

## Purpose

This section is not meant to showcase full applications, but to:

- Reinforce core concepts through hands-on coding
- Serve as reference implementations
- Support the development of larger AI projects in the `projects/` folder

## Structure

Each subfolder contains:

- Small, focused Python scripts
- Minimal dependencies
- Clear examples of usage

## Note

As the focus shifts toward building real-world AI systems, most of the work will move into the `projects/` directory.

This folder represents the foundation that supports those systems.
